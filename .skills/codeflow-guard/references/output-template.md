- 静态发现跳过测试：0（来自 `test.skip` / `describe.skip` / `it.skip`）

## 复审标准

复审时必须重新运行上下文采集脚本和必跑测试。只有同时满足以下条件，才能把合并建议提升为“可合并”：

- P0 = 0
- P1 = 0
- 必跑测试通过
- 关键测试跳过数量 = 0
- diff 中不存在硬编码密钥、令牌、密码或敏感连接串
